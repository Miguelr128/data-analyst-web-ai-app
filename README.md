# 🤖 AI Data Analyst Web

An intelligent data analysis platform that combines the power of **Google Gemini Flash 1.5** with **FastAPI** and **React** to transform raw CSV data into executive narratives and interactive visualizations.

## 🚀 Features

- **Intelligent Ingest**: Extracts schema and sample data to minimize token usage and optimize cost.
- **Narrative Insights**: Executive-level summaries generated by AI in English.
- **Interactive Charts**: Recharts-powered visualizations for easy data exploration.
- **Supabase Integration**: 
  - **Execution Logging**: Every analysis is logged in the `task_logs` table.
  - **Error Tracking**: Technical failures are captured and cataloged in the `error_logs` table.
- **Jupyter Export**: Generate and download a complete `.ipynb` notebook for further research.
- **Mock Mode**: Demonstrates UI capabilities even without an active API key.

## 🛠️ Tech Stack

- **Backend**: Python, FastAPI, Google Generative AI (Gemini).
- **Frontend**: Vite, React, Tailwind CSS, Recharts.
- **Database/Storage**: Supabase (PostgreSQL).

## ⚙️ Setup & Installation

### 1. Prerequisites
- Python 3.9+
- Node.js & npm
- A Google Gemini API Key
- A Supabase Project (URL and Anon key)

### 2. Backend (FastAPI)
1. Navigate to the `api` folder.
2. Create a `.env` file based on `.env.example`:
   ```env
   GEMINI_API_KEY=your_gemini_key
   SUPABASE_URL=your_supabase_url
   SUPABASE_ANON_KEY=your_anon_key
   PORT=8000
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the server:
   ```bash
   python main.py
   ```

### 3. Frontend (React + Vite)
1. Navigate to the `client` folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser.
